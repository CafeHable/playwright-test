# Playwright 자동화 테스트 포트폴리오

## 소개
Playwright를 활용한 웹 자동화 테스트 프로젝트입니다.

## 사용 기술
- Playwright
- JavaScript
- Node.js

## 테스트 목록
| 테스트 | 설명 | 결과 |
|---|---|---|
| 로그인 성공 테스트 | 올바른 아이디/비번으로 로그인 성공 확인 | ✅ PASS |
| 로그인 실패 테스트 | 잘못된 아이디/비번으로 에러 메시지 확인 | ✅ PASS |

## 실행 방법
```bash
npm install
npx playwright test
```

## 테스트 결과
- 총 2개 테스트
- 2/2 통과 (100%)
